Posted on 2020-06-03 Edited on 2022-01-04 In MISC

visual studio code 插件：

  1. Markdown All in One：实时预览工具
  2. Markdown Emoji：让 markdown 支持 emoji
  3. Markdown PDF：将 md 转 pdf
  4. markdownlint：格式检查

chrome 插件：

  1. Markdown Viewer：让 chrome 支持 md 文件实时预览
